Welcome to LungoJS KitchenSink

The first Mobile Framework that uses the actual features of #HTML5, #CSS3 and #JavaScript.

The main premise in Lungo is to create a semantic structure in the whole project, starting from the markup language HTML, through a well organized CSS< and ending with the JavaScript API

The data-attributes

  • data-icon Creates an icon LungoJS vectorized
  • data-image Instance a <img> automatically
  • data-search Creates a search input form
  • data-progress Creates a progress element

Others

  • data-title Relationship field between <header> and <a>
  • data-count Creates counters of different <elements>
  • data-back Creates a button that goes to the previous section.
